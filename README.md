# 🚗 ParkWise

ParkWise is a smart parking app that helps drivers quickly find and navigate to available parking spots inside a parking lot.

---

## 🎯 Features

* Real-time parking availability (mocked)
* Parking spot assignment
* Simple in-parking navigation
* Active parking session with timer
* Issue reporting
